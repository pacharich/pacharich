▼ 開き方
「モデル候補リスト.html」をブラウザ（Chrome / Safari / Edge 等）で開いてください。
各モデルカードをクリックすると、宣材写真の一覧と詳細プロフィールがご覧いただけます。

▼ ファイル構成
- モデル候補リスト.html       ... メインのHTMLファイル
- app.js                      ... モデル一覧データとUI制御
- styles.css                  ... スタイルシート
- image-slot.js               ... 画像スロット用コンポーネント
- .image-slots.state.json     ... 取り込み済み写真データ

▼ 注意点
※ ファイルは必ずすべて同一フォルダ内に配置してください（個別に切り離して使用しないでください）
※ 画像データは .image-slots.state.json に内包されています。フォルダごとお送りください。
※ 隠しファイル（.image-slots.state.json）を圧縮 / 解凍ソフトで除外しないようご注意ください。
